Module: components/form/ToggleComponent

A component to put a toggle button in a form

Props

Name Type Default value Required? Description
symbols String|Array.<String> ["-", "+"] No Strings to use to represent the values of the toggle. The symbol indexes should map the to vals indexes
vals Array [true,false] No values the toggle toggles to. Toggling increases the value index by 1 and then loops around to 0
val Boolean|String|Number|Object - Yes the current value of the toggle
translate Boolean false No should the symbols be translated?
clear Boolean|String false No should toggling also clear a field on the form? True clears the field of the same name as the toggle, a string will clear the field at the given path
field String - Yes the field that is being toggled
clearOn Array - No an array of values that the should clear the field supplied by clear. If this property is not supplied, all toggles will clear the clear field
expandable Boolean false No does this toggle expand a menu?

Computed

Name Type Description
toggleState String the label taken from symbols to describe the toggle's current state
index Number the index the current value has in the vals array
expanded Boolean is the menu this toggle expands exapnded?
expanded Boolean should screen readers read this button as pressed?
