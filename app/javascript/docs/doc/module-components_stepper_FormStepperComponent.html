Module: components/stepper/FormStepperComponent

A component to make multi-step forms. Uses Flickity to handle form step navigations

Data

Name Type Default value Description
curIndex Number 0 the current step index
internalInd Number 0 the index of the current step in the $children array
indexes Array.<Number> [] an array mapping the step indexes to their internal indexes
numSteps Number 0 the number of steps in the stepper
completes Array.<Boolean> - an array to keep track of which steps have been visited and completed
curStepPending Boolean - is the current step not yet ready?
FlickityOptions Object - the options for the flickity instance
