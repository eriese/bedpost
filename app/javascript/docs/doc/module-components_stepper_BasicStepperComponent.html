Module: components/stepper/BasicStepperComponent

A component that allows you to put steps on a page and animate navigation

Props

Name Type Default value Required? Description
animate Boolean true No whether step transitions should animate
defaultProps module:transitions.prop_overrides - No default animation props to apply to animations

Data

Name Type Default value Description
step number 0 the step the stepper is currently showing

Mixes In:
  • renderless
Example

Put the stepper in your html using a scoped slot. Then use v-ifs to separate your steps, and call module:components/basicStepperComponent.setStep to change steps

<basic-stepper :default-props="{animLength: 0.2}">
 <div slot-scope={step, setStep}>
		<div v-if="step == 0">
			I'm the first step
			<button type="button" @click="setStep(1, {animationType: 'slideLeft'})">Go to the second step</button><
		</div>
		<div v-if="step == 1">
			I'm the second step
			<button type="button" @click="setStep(0, {animationType: 'slideRight'})">Go back to the first step</button>
		</div>
	</div
</basic-stepper>
