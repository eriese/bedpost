Module: components/basicStepperComponent

A component that allows you to put steps on a page and animate navigation

Source:

Examples

It's easiest to put the stepper in your html with an inline template. Then use v-ifs to separate your steps, and call module:components/basicStepperComponent.setStep to change steps

<basic-stepper inline-template :default-props="{animLength: 0.2}">
	<div v-if="step == 0">
		I'm the first step
		<button type="button" @click="setStep(1, {animationType: 'slideLeft'})">Go to the second step</button><
	</div>
	<div v-if="step == 1">
		I'm the second step
		<button type="button" @click="setStep(0, {animationType: 'slideRight'})">Go back to the first step</button>
	</div>
</basic-stepper>

You can also put it on the page using a scoped slot.

<basic-stepper>
	<div slot-scope={step, setStep}>
		... see above example
	</div>
</basic-stepper>
