Module: components/stepper/FormStepComponent

A component for a form step. Works with parent component FormStepperComponent

Props

Name Type Default value Required? Description
optional Boolean false No is this step optional to complete?
numSteps Number - Yes the number of steps the parent component has as children

Data

Name Type Default value Description
index Number 0 the index this step has in the list of steps
completes Array.<Boolean> [] an array containing the completeness values for each field in the step

Computed

Name Type Description
fields Array.<module:components/fieldErrorsComponent> the validated fields that are children of this step

Listens to Events:
  • module:components/form/FieldErrorsComponent~event:input-blur
Example

Use a form step to wrap a form field or fields in a scoped slot. Don't forget to attach the blur event or have it called from somewhere in your component.

<form-step :num-steps="stepper.numSteps">
	<div slot-scope="step">
		<input @blur="step.fieldBlur">
	</div>
</form-step>
